# Getting started

## Prerequisites

- [NodeJS](https://nodejs.org/)
- [Npm](https://docs.npmjs.com/)
- [Vite](https://vite.dev/guide/)

```markdown
npm install -g vite
```

## *Start the local dev environment*

```markdown
npm run dev
```

## *Compile project*

```markdown
npm run dev
```

## *Preview build project*

```markdown
npm run preview
```

## *Run biomeJS*

```markdown
npm run check
```
